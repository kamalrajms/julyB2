## react intro

    What is React?

    1) React is a JavaScript library (not a framework) developed by Facebook (Meta) for building user interfaces (UIs), especially Single Page Applications (SPAs).

    2) It helps developers build reusable UI components that update efficiently when data changes.

    Instead of reloading a whole webpage when something changes, React updates only the parts of the UI that need to change.

    -------
    Why do we need React?

	It makes UI development faster, cleaner, and more efficient.

	Provides a better user experience with smooth updates without page reloads.

## useState()

    useState is a react hook that allows you to add state (data that can change over time)
    to a function component.

    Normally, in js function , variables are disapper after the function runs, but in 
    with useState, react remembers values b/w re-render.


    syntax:


        const [state,setState]=useState(initialValue)

        state-- the current value of the state variables
        setState--- a function that update the state
        initialValue---  the starting value of the state.

        eg:

            const count=20;

## useEffect()

     In react ,useEffect is a hook that lets you perform side effect in function 
     component.
     Side effect are task that happen outside the normal react rendering process , like:


            fetch data from API
            updateing the browser dom manually
            strting/stoping timers


    syntax:

        useEffect(()=>{
            //code
        },[dependency 1,dependency 2....])

        [] -- it will execute only once at initially.


## useRef()

    useRef is a React hook that gives you a box (.current) you can store anything in
    --like num,str,obj,function,even dom element.

    That value persists b/w render

    Updating it does not cause your component to re-render.

    {current:value}


    syntax:

        const variables=useRef(initialValue)
